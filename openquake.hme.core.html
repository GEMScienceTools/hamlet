openquake.hme.core.core module¶

Core functions for running Hamlet.

The functions here read the configuration file, then load all of the model inputs (seismic sources, observed earthquake catalog, etc.), run the tests, and write the output.

openquake.hme.core.core.get_test_lists_from_config(cfg: dict) → dict¶

Reads through the cfg and makes a dict of lists of tests or evaluations to run for each framework.

Parameters

cfg – Configuration for the evaluations, such as that parsed from the YAML config file.

openquake.hme.core.core.load_inputs(cfg: dict) → Tuple[geopandas.geodataframe.GeoDataFrame]¶

Loads all of the inputs specified by the cfg and returns a tuple of GeoDataFrame objects, the earthquake catalog and the bins.

Parameters

cfg – Configuration for the evaluations, such as that parsed from the YAML config file.

openquake.hme.core.core.load_ruptures_from_ssm(cfg: dict)¶

Reads a seismic source model, processes it, and returns a GeoDataFrame with the ruptures. All necessary information is passed from the cfg dictionary, as from a test configuration file.

Parameters

cfg – Configuration for the evaluations, such as that parsed from the YAML config file.

Returns

A GeoDataFrame of the ruptures.

openquake.hme.core.core.run_tests(cfg: dict) → None¶

Main Hamlet function.

This function reads the cfg, loads all of the inputs, runs the evaluations, and then writes the ouputs.

Parameters

cfg – Configuration for the evaluations, such as that parsed from the YAML config file.

openquake.hme.core.core.write_outputs(cfg: dict, bin_gdf: geopandas.geodataframe.GeoDataFrame, eq_gdf: geopandas.geodataframe.GeoDataFrame, write_index: bool = False) → None¶

Writes output GIS files and plots (i.e., maps or MFD plots.)
